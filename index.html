Repositório CMD - Scripts

Este site hospeda scripts PowerShell para download e execução.

Como Funciona:

Este script PowerShell automatiza o download e a extração dos arquivos do repositório GitHub para o seu computador.

Para executar o script diretamente no seu sistema, abra o Prompt de Comando (CMD) ou PowerShell e cole o comando abaixo:

powershell -c "iwr https://raw.githubusercontent.com/gu2121gg/cmd/main/script.ps1 | iex"

⚠️ **Atenção:** Ao executar scripts da internet, sempre verifique o conteúdo para garantir a segurança. Este script é fornecido como exemplo e deve ser usado com cautela.

Conteúdo do Script PowerShell (`script.ps1`):

$repoUrl = "https://github.com/gu2121gg/cmd/archive/refs/heads/main.zip"; # URL direta para o ZIP do seu repositório
$destinationFolder = ".\cmd_repo_download"; # Pasta onde o repositório será extraído (no diretório atual)
$tempZipFile = Join-Path $env:TEMP "cmd_repo_temp.zip"; # Caminho para o arquivo ZIP temporário

# Cria a pasta de destino se ela não existir, forçando a criação se necessário
New-Item -ItemType Directory -Force -Path $destinationFolder | Out-Null;

Write-Host "Iniciando download e extração do repositório GitHub...";

try {
    # Baixa o arquivo ZIP do repositório. Out-Null evita output desnecessário.
    Invoke-WebRequest -Uri $repoUrl -OutFile $tempZipFile -ErrorAction Stop | Out-Null;

    Write-Host "Arquivo ZIP baixado. Extraindo conteúdo...";

    # Extrai o conteúdo do ZIP para a pasta de destino. O parâmetro -Force sobrescreve se existir.
    Expand-Archive -Path $tempZipFile -DestinationPath $destinationFolder -Force -ErrorAction Stop;

    Write-Host "Conteúdo extraído. Limpando arquivos temporários...";

    # Remove o arquivo ZIP temporário. SilentlyContinue evita erros se o arquivo já não existir.
    Remove-Item -Path $tempZipFile -Force -ErrorAction SilentlyContinue | Out-Null;

    # --- Nova funcionalidade: Abrir index.html ---
    # Nota: Este script PowerShell está embutido em um HTML e o objetivo é que ele baixe e extraia
    # o conteúdo do seu repositório. A funcionalidade de "abrir index.html" abaixo se refere a
    # tentar abrir um index.html *dentro do conteúdo baixado*, não o HTML que hospeda este script.
    $extractedContentPath = Get-ChildItem -Path $destinationFolder -Directory | Where-Object { $_.Name -like "cmd-main*" }; # Ajustado para encontrar a pasta extraída (geralmente 'cmd-main')

    if ($extractedContentPath) {
        $indexPath = Join-Path $extractedContentPath.FullName "index.html";
        if (Test-Path $indexPath) {
            Write-Host "Abrindo index.html no navegador...";
            Start-Process $indexPath; # Abre o arquivo no programa padrão (navegador para .html)
        } else {
            Write-Warning "index.html não encontrado na pasta extraída: $extractedContentPath.FullName";
        }
    } else {
        Write-Warning "Não foi possível localizar a pasta do repositório extraída em: $destinationFolder";
    }
    # --- Fim da nova funcionalidade ---

    Write-Host "Processo concluído.";

} catch {
    Write-Error "Ocorreu um erro durante o processo: $($_.Exception.Message)";
    # Se houver um erro, tenta limpar o ZIP temporário
    if (Test-Path $tempZipFile) { Remove-Item -Path $tempZipFile -Force -ErrorAction SilentlyContinue | Out-Null; }
}
